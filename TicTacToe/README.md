A simple cross platform (iOS and Android) React Native ToDoList app.

### Installation

- `git clone https://github.com/anil843324/LearningReactNative/tree/master/ToDoList`
- `npm install`

### Running

- `npx react-native run-ios` or `npx react-native run-android`
